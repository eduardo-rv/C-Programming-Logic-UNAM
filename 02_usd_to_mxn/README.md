# Program 02: Currency Converter (USD to MXN)

## 📝 Assignment
Develop a C program that performs a currency conversion from United States Dollars (USD) to Mexican Pesos (MXN). The program should demonstrate the use of modular programming by utilizing functions for input handling, calculation, and display.



## 🛠 Requirements
1. **Functional Design:**
     Create a function to handle the user input for the amount.
     Create a specific function to perform the math and display the result.
2. **User Input:** Prompt the user to enter the amount in USD (using floating-point numbers for precision).
3. **Logic Implementation:** Use a fixed exchange rate to convert the currency.
4. **User Interface:** Include a clear header and ensure the terminal is cleared before starting the main process.
5. **Data Formatting:** Output the conversion result formatted to 2 decimal places.



## 📐 Mathematical Formula
The conversion is based on a fixed rate (e.g., $18.00$ MXN per $1.00$ USD):

$$Result (MXN) = Amount (USD) \times Exchange Rate$$



## 🚀 Challenges
1. **Dynamic Exchange Rate:** Modify the program to ask the user for the *current* exchange rate instead of using a hardcoded value (like 18).
2. **Bi-directional Conversion:** Add a menu that allows the user to choose between converting USD to MXN or MXN to USD.
